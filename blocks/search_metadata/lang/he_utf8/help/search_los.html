Search LOs-metadata (Learning Objects)
Resources/Activities

This block allows to search all the resources and activities, inside a course, in the user courses, or in all the courses, since the resources have been previously annotated with metadata LOM or metadata DCMES. The basic search supports a number of options, listed below. It is possible to combine them to specify your search more precisely.

search for these words For basic searching of one or more words anywhere in the texts, just type them separated by spaces. All words longer than two characters are used.
+search +for these words The previous example would also match "information" because it contains "for". To force exact matching of a word, use the plus sign.
+search -engine Use the minus sign (-) if there are particular exact words you don't want included in the search.
"search engine" To search for a particular phrase, use double quotes around it.
user:Vitor To search for resources/activities by a particular user, prefix a word from their name with "user:".
userid:6 If you know the user id of a particular user, you can search for them like this.
subject:program To search for a word within the subject or title of a text only, prefix the word with "subject:".

For advanced searching, press the search button ( > ) without typing anything in the words field - you will see a complete form that makes it easier to do advanced searches.

In the context of the advanced research, you will see:

Metadata: You can select the type of metadata for searching (LOM or DCMES)
Courses: You can specify what course(s) search inside a course, in the user courses, or in all the courses.
LOs type:

The block will be able to search only a specific type of resources/activities, since theresource/activity metadata refers it.

NOTE: The resources and activities that have not been annotated (automatically or not by the teacher) with LOM metadata or DCMES metadata will not be available for the research of LOs (Learning Objects - resources or activities).
