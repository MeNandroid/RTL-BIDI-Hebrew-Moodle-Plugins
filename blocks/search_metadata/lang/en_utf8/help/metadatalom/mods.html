LOM Metadata

Metadata is “data about data”. Metadata is structured data set that describes, explains and facilitates the localization and recovery of information.
LOM (Learning Object Metadata) specifies a conceptual data schema that defines the structure of a metadata instance for a learning object. For this Standard, a learning object is
defined as any entity -digital or non-digital- that may be used for learning, education or training. (LOs - Learning Objects). It is desirable that the metadata creation occurs when the LO (resource/activity) creation and that the LO’s author is also the author of the LO metadata.

The elements that describe an LO can be grouped in nine categories [IEEE-FTSC 2002]:

General - This category groups the general information that describes this learning object as a whole. (Elements: Identifier, Title, Catalog Entry, Catalog, Entry, Language, Description, Keywords, Coverage, Structure, e Aggregation level);

Lifecycle - This category describes the history and current state of this learning object and those entities that have affected this learning object during its evolution. (Elements: Version, Status, Contribute, Role, Entity, Date);

Metametadata - This category describes this metadata record itself (rather than the learning object that this record describes). Describes how the metadata instance can be identified, who created this metadata instance, how, when, and with what references (Elements: Identifier, Catalog, Entry, Contribute, Role, EntitytDate, Metadata Squema, Language);

Technical - This category describes the technical requirements and characteristics of this learning object. (Elements: Format, Size, Location, Requirements, Type, Name, Minimum Version, Maximum version, Instalattion Remarks, Other Plataform requirements, Duration);

Educational - This category describes the key educational or pedagogic characteristics of this learning object. (Elements: Interactivity Type, Learning Resource Type, Interactivity level, Semantic density, Intended End User Role, Context, Typical Age Range, Difficulty, Typical Learning Time, Description, Language);

Rights - This category describes the intellectual property rights and conditions of use for this learning object.(Elements: Cost, Description, Copyright and others restrictions);

Relation– This category defines the relationship between this learning object and other learning objects, if any. (Elements: Kind, Resource, Identifier, Description, Catalog Entry);

Annotation - This category provides comments on the educational use of this learning object, and information on when and by whom the comments were created. (Elementos: Person/Entity, Description, Date);

Classification - This category describes where this learning object falls within a particular classification system, like UDC (Universal Decimal Classification) or ARIADNE, etc. (Elements: Purpose, Taxon Path, Source, Taxon, Id, Entry, Description, Keywords).

For more information see: http://ieeeltsc.org/wg12LOM/
