LOM Metadata
     LOs (Learning Objects)
  • Name: Insert a name or short title that characterizes the LO that you want annotate.
  • LO (Learning Object): The resources or activities are the LOs (Learning Objects) of a Moodle course. Choose the resource or activity of the actual course which intends to add LOM metadata. The resources or activities that already have been annotated with LOM do not appear in this combo box. Notice: the resources or activities eventually annotated with DCMES will continue appear in this box, allowing that the same LO can be annotated with two different specifications.
  • "Next" Button: When you insert a new LOM metadata, click this button and some of DC elements will automatically be generated in accordance with the resource or activity selected.
  • "Refresh metadata automatically" Button: When you edit or update LOM metadata, click this button and the elements or fields that are null will be filled with the metadata automatically generated and the combo boxes with the option “insert new value” selected are transformed into input text fields..
