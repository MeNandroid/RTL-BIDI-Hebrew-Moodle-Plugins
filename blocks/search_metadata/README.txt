Search LO-metadata block for Moodle - version 1.0 (demo release)
===============================================================================
Created by:
      Vitor Gonçalves - coding & design

Developed for Escola Superior de Educação - Instituto Politécnico de Bragança -
Portugal and Faculdade de Engenharia da Universidade do Porto (academic project)

Many ideas and code were taken from other Moodle modules and Moodle itself...

Installation:
    1/ Verify your Moodle installation. Recommended version: 1.6.x or later
       (but it will work without problems in early versions)
    2/ download search_metadata.zip to your moodle/blocks directory and unzip it there
    3/ go to http://yoursite.com/admin - the block will be instaled...


List of features - version 1.0:

    * easy to use - teachers and pupils can quickly search their resources/activities
	annotated with metadata DC or metadata LOM
    * works with MySQL database
