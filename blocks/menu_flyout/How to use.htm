How to use the Menu Flyout

The instructions in this document assumes that

  • You have a Moodle course built or that you have a course structure in mind
  • Your course is using the Moodle topics course format and that you have the topics that you want to link to already available to use
  • Your course has a structure that benefits from a menu for users to navigate the information
  • You have an understanding of what a 'flyout menu' is and how best to use one

*If your course is not set to the topics course format, please change this before attempting to create a menu for your course*

How do I insert a menu?
  1. Once inside your course, 'turn on editing' using the top right button
  2. Scroll to the bottom of the right-hand column to the section labelled 'Blocks'
  3. Select 'Menu' from the dropdown list, your new Menu block should now appear
  4. Move your menu block to where you feel it would be most appropriate for your course

 

How build my menu?

  1. On you Menu block, click the 'Menu Maker' link. You will be taken to the menu editor screen
  2. Follow the steps to create your menu
    • Step 1 - Choose a topic number. This is the number of the topic that you would like this link to jump too
    • Step 2 - Give this link a title. This is what the user will see so give it a short desciptive name that the user will find helpful
    • Step 3 - Choose wether this link will be a main link, or a sublink of another link. links are able to go 3 sublinks deep. For example, 'Files' could be a sublink of a catergory link called 'Folder' which could be contained under a main link called 'Cabinet'.
    • Repeat steps 1 - 3 for however many links you need in your menu. If you make a mistake, you are able to remove individual link from the menu example at the bottom of the page.
      Please note that you can only insert the topic into the menu as a link once.
  3. Once you are happy with your menu structure click on the link at the bottom of the page labelled 'Back to Course' to return to your course and your new menu.

 

What if I need to re-edit my menu?
  1. Click on the 'Turn editing on' button at the top right of your screen
  2. Find your 'Menu' block and click the 'Menu Maker' link at the bottom of the block
  3. You will be taken to the menu editor page where you can re-edit your menu

 

How do I add my own styles to the menu?
  1. Go to the course files and create a new folder named 'do_not_delete' (all in lowercase)
  2. Inside this folder create a files called 'flyoutmenu.css', or you can grab a copy from the folder '{moodle directory}\blocks\menu_flyout\flyoutmenu.css' and save it in the 'do_not_delete' folder. (Do not remove the 'flyoutmenu.css from this folder though, as it is the default styles for the menu, site-wide!)
  3. Once added to the do_not_delete folder, edit or add the styles that you want to 'flyoutmenu.css' (CSS knowledge required). Don't worry about things going wrong, if you ever need to you can delete the 'flyoutmenu.css' from the 'do_not_delete' folder and the menu will revert back to the original style.

 

How do I uninstall the menu block and editor?
  1. To remove menu editor, simply locate the menu_flyout folder inside '{moodle directory}\blocks\' and delete it. Menu editor will no long be available on any of your courses.
