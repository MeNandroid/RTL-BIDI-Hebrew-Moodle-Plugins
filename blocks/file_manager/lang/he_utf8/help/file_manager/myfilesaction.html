מנהל קבצים

פעולות על קישורים

הזזה לתיקיה אחרת
ציין את הקישורים או הקבצים שברצונך להזיזים ולאחר מכן בחא את הפעולה "הזז לתיקיה אחרת" מהרשימה. הסתעף לאזור המטרה הסופי והקש על הפעולה "הזז קבצים לכאן".

שיוך לקטגוריה
בחר קישורים או קבצים שברצונך לשיך לקטגוריה ולאחר מכן הקש על הפעולה "שיך קטגוריה", ברשימה ובחר את שם הקטגוריה לפני שאתה מקיש על כפתור "שייך".

בנית ארכיב ZIP
בחר קישורים או קבצים שאתה רוצה להוסיף לארכיב. בחר את הפעולה "בנה ארכיב ZIP" ולבסוף בחר את צורת ה-ZIP: סטנדרטית או ZIP של מוודל.

מחיקה
בחר את הקבצים או הקישורים שברצונך למחוק ואז בחר את הפעולה "מחק" מהרשימה. Select files or links that you want to delete and select the action "Delete" in the list.
