LOM Metadata
     Relation

This category defines the relationship between this learning object and other learning objects, if any.

  • Kind: Nature of the relationship between this learning object and the target learning object, identified by Relation.Resource.
  • Resource: The target learning object that this relationship references.
    • Identifier: A globally unique label that identifies the target learning object.
      • Catalog: The name or designator of the identification or cataloging scheme for this entry. A namespace scheme.
      • Entry: The value of the identifier within the identification or cataloging scheme that designates or identifies the target learning object. A namespace specific string.
    • Description: Description of the target learning object.

Note:
a) 0-1 (0 minimum elements - 1 maximum elements) or 1-n (1 minimum elements - n maximum elements).
b) To add more than one value use ";; " to separate the values.
c) You can choose the option “Insert new value” to transform a combo box into input text box and click button “Refresh metadata automatically”.
d) In a combo box, choose the option “ “(empty or null) to not use values to the DC element.
e) If you want restore the predefined values, cleans the fields in question, save and come back to edit the register of metadata so that the values are restore automatically.
