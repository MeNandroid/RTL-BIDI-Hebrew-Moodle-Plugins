Grade to be used

You can choose any available graded activities to print the grade received for that activity on the certificate and block certificate emission. If you choose to print the Course Grade, the mean of all grades for this course will be printed and will be used to block certificate emission. Please note that grade exclusions are not taken into account for the course grade.

Grade Format

There are three available formats if you choose to print a grade on the certificate:

Percentage Grade: Prints the grade as a percentage.
Points Grade: Prints the point value of the grade.
Letter Grade: Prints the percentage grade as a letter. The values for the letter grades is the same of course settings.
