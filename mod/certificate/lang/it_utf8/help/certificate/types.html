Tipo di Certificato

È qui che puoi determinare l'aspetto del'attestato. Vi sono alcuni modelli di base degli attestati nella cartella "certificate/type" ( ove sono contenuti tutti i modelli ).

Orizzontale (A4): stampa sul formato A4 con orientamento orizzontale.
Orizzontale (letter): stampa sul formato letter con orientamento orizzontale.
Verticale (A4): stampa sul formato A4 con orientamento verticale.
Verticale (letter): stampa sul formato letter con orientamento verticale.
Unicode (Orizzontale): stampa in formato UniCode con orientamento orizzontale.

Nuovi modelli possono essere aggiunti nella cartella "certificate/type".
I nomi delle cartelle e le stringhe relative alla lingua di ogni nuovo modello devono essere aggiunte al file "certificate/lang/en_utf8/certificate.php" ed ai files corrispondenti ad ogni lingua.
