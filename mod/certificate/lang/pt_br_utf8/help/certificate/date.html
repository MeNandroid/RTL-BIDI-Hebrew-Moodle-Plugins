Imprimir data

Seleciona se uma data deverá ser impressa no certificado. Se você escoher imprimir a data ddo final do curso você deverá ajustar uma data para o final do curso nas configurações do curso.

Note que se você optar que uma vez que uma data seja atribuída a um certificado ela não poderá ser modificada a não ser que você tenha customizado seu arquivo type/certificate.php.

Formato da data

Escolha o formato de data que você deseja que seja usado em seu certificado.
