รูปแบบวันที่

เลือกรูปแบบวันที่ที่จะพิมพ์ลงในประกาศนียบัตร:

%DD - วันที่ 2 หลัก ถ้ามีหลักเดียวจะเติม 0 ไว้ข้างหน้า

%dd - วันที่แบบไม่เติม 0 ไว้ข้างหน้า

%ss - ลำดับของวันที่ในภาษาอังกฤษ 2 ตัวอักษร เช่น st, nd, rd

%MS - ชื่อเติมของเดือน

%mm - เลขลำดับที่ของเดือนแบบไม่เติม 0 ไว้ข้างหน้า

%MM - เลขลำดับที่ของเดือนแบบเติม 0 ไว้ข้างหน้า

%YY - เลขปีแบบ 4 หลัก

%yy - เลขปีแบบ 2 หลัก

ตัวอย่าง:

%DD %MS %YY = 05 มกราคม 2007

%DD/%MM/%YY = 05/01/2007

%dd/%mm/%yy = 5/1/07

วันที่ %DD เดือน%MS ค.ศ. %YY = วันที่ 05 เดือนมกราคม ค.ศ. 2007
