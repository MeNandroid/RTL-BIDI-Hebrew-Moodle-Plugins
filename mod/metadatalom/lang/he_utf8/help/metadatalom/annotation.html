LOM Metadata
     Annotation

This category provides comments on the educational use of this learning object, and information on when and by whom the comments were created.

  • Entity: Entity (i.e., people, organization) that created this annotation. Use vCard. You don't need to add "BEGIN:VCARD" neither "END:VCARD".
  • Date: Date that this annotation was created.
  • Description: The content of this annotation.

Note:
a) 0-1 (0 minimum elements - 1 maximum elements) or 1-n (1 minimum elements - n maximum elements).
b) To add more than one value use ";; " to separate the values.
c) You can choose the option “Insert new value” to transform a combo box into input text box and click button “Refresh metadata automatically”.
d) In a combo box, choose the option “ “(empty or null) to not use values to the DC element.
e) If you want restore the predefined values, cleans the fields in question, save and come back to edit the register of metadata so that the values are restore automatically.

For more information see: http://ieeeltsc.org/wg12LOM/
