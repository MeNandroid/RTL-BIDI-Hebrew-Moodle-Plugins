Metadata LOM module for Moodle - version 1.0 (demo release)
===============================================================================
Created by:
      Vitor Gonçalves - coding & design

Developed for Escola Superior de Educação - Instituto Politécnico de Bragança -
Portugal and Faculdade de Engenharia da Universidade do Porto (academic project)

Many ideas and code were taken from other Moodle modules and Moodle itself...

Installation:
    1/ Verify your Moodle installation. Recommended version: 1.6.x or later
       (but it will work without problems in early versions)
    2/ download metadatalom.zip to your moodle/mod directory and unzip it there
    3/ go to http://yoursite.com/admin - the necessary table will be created and
	 log_display will be updated


List of features - version 1.0:

    * easy to use - teachers can quickly add metadata for their resources/activities
    * semi-automatic fields filled with existing data or or in accordance with the
	type of resource
    * support for global searching - supported with Search LOs-metadata block
    * works with MySQL database

    * works with PostgreSQL database - not supported yet
    * backup/restore/import -  not supported yet
    * Language packs included - en_utf8, pt_utf8, fr_utf8
