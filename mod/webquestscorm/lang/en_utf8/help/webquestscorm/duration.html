Duration

The format of the duration element shall be represented in accordance with ISO8601:2000:

P[yY][mM][dD][T[hH][nM][s[.s]S]]

Where:
  • y = number of years
  • m = number of months
  • d = number of days
  • h = number of hours
  • n = number of minutes
  • s = number of seconds or fraction of seconds

The character literal designators P , Y, M, D, T, H, M and S must appear if the corresponding nonzero value is present.

Example: PT1H30M
