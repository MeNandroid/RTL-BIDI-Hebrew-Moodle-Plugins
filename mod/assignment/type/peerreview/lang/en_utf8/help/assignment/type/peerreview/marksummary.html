Marks and Mark Summary

Marks are awarded for completing reviews and also for achieving set criteria. The total of these needs to sum to the final grade for the assignment. This page will attempt to enforce a correct total for the final grade and advise you if marks need to be added or removed.

The final grade and the reward for each of the two reviews are set with the assignment settings. If you would like to change these settings click the 'Update this Peer Review Assignment' button above then come back and set the criteria marks.
