Comment System

  • Default
    Uses the site wide setting.
  • No comment system
    This wiki does not use a comment system.
  • Annotation system
    A user, with the appropriate permission, can add inline annotations to the wiki.
  • Per-section comment system
    This is the original method of commenting whereby the comments are added at the top of each section.
  • Both sytems
    Both of the above systems can be used on the same wiki.
